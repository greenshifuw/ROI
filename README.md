
# Simulateur ROI MyFanjan

Application web pour simuler le Retour sur Investissement (ROI) des tours hydroponiques MyFanjan par rapport à la culture en pleine terre.

## Structure
Ce projet est un site statique composé de :
- `index.html` : Structure de la page.
- `style.css` : Styles et design (Responsive).
- `script.js` : Logique de calcul (réplique les formules Excel).

## Déploiement

### Vercel
1. Installez Vercel CLI ou connectez votre compte GitHub à Vercel.
2. Importez ce dossier comme "Static Project".
3. Déployez. Aucune configuration de build n'est nécessaire.

### GitHub Pages
1. Activez GitHub Pages dans les paramètres du dépôt.
2. Choisissez le dossier (root) comme source.

## Fonctionnalités
- **Comparatif** : Pleine Terre vs Hydroponie.
- **Paramètres** : Tous les intrants (Hypothèses) sont modifiables.
- **Rapports** : 
    - Synthèse KPI.
    - Compte d'exploitation détaillé.
    - Plan de trésorerie sur 5 ans.
